# CineBook - Movie & Book Directory 📚🎬

CineBook adalah aplikasi direktori digital yang dirancang untuk mengelola dan menjelajahi Mobile Katalog Film serta Buku dalam satu platform yang dibangun menggunakan **flutter**. proyek ini merupakan pemenuhan tugas UAS dengan fokus, antarmuka yang elegan dan fungsional pencarian dinamis.

Aplikasi ini tidak hanya fokus pada fungsionalitas, tetapi juga pada pengalaman pengguna (UX) dengan menerapkan desain minimalis dan responsif.
 
---

## Daftar Fitur Saat Ini
1. **Dual Directory :** Memisahkan katalog Film dan Buku melalui navigasi Tab.
2. **Real-time Search :** Memungkinkan pengguna mencari judul secara langsung melalui kolom pencarian.
3. **Favorite system (Persistence) :** Pengguna bisa menandai favorite yang datanya tetap tersimpan meski aplikasi ditutup (menggunakan 'SharedPreferences').
4. **Detail View :** Menampilkan informasi lengkap mulai dari poster besar, Judul, rating, kategori hingga  deskripsi lengkap.
5. **Dark Mode UI :** Menggunakan tema gelap yang nyaman di mata dengan akses warna amber.

## Teknologi yang digunakan
1. **Language :** Dart
2. **Framework :** Flutter (Material 3)
3. **State Management :** Provider
4. **Local Storage :** Shared Preferences
5. **version Control :** Git & GitHub

## Cara Menjalankan Proyek
1. Pastikan **Flutter SDK** sudah terinstal di perangkat anda.
2. Clone reposetory ini.
3. Jalankan perintah 'flutter pub get' untuk mengunduh dependencies.
4. Gunakan perintah 'flutter run' untuk menjalankan di emulator/perangkat asli.
